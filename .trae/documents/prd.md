## 1. 产品概述

茶席器具摆位试排工具——帮助茶艺师在矩形席面内预排器具位置，实时校验合规性并计算最短动线步数，避免实际摆位时因动线不合理被客人挡路。

- 目标用户：茶艺培训学员、茶席设计师、茶道爱好者
- 核心价值：零成本试错、即时反馈合规与动线、一键导出摆位方案

## 2. 核心功能

### 2.1 功能模块

1. **席面摆位页**：6×4 网格席面、器具侧栏、合法性校验、动线步数、导出摆位表

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 席面摆位页 | 席面网格 | 6×4 矩形网格，入口格（左下角）与出口格（右下角）标注，空格可放置器具，已放置格可点击清空 |
| 席面摆位页 | 器具侧栏 | 四种器具（壶/公道/杯/盂）点选切换当前选中器具，显示已放置/待放置状态 |
| 席面摆位页 | 合法性校验 | 实时检测三条规则，违规项红色提示，合规项绿色勾选 |
| 席面摆位页 | 动线步数 | 四件均落位且合法时，计算并展示最短动线步数 |
| 席面摆位页 | 导出摆位表 | 四件均落位且合法时按钮可用，输出 JSON 含器具代号、行列、最短动线步数 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖茶色系（#8B6914 深茶褐、#D4A574 浅驼色）、墨色（#2C2C2C）文字
- **辅色**：竹绿（#6B8E6B）表示合规、朱红（#C94040）表示违规
- **按钮风格**：圆角微凸、哑光质感
- **字体**：衬线展示字体（Noto Serif SC）+ 无衬线正文（Noto Sans SC）
- **布局风格**：居中卡片式布局，左侧器具侧栏，中央网格，右侧校验/导出面板
- **图标风格**：简约线性图标，配合茶器意境

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 席面摆位页 | 席面网格 | 6×4 网格卡片，浅木色背景，入口/出口格特殊标记，器具占格显示器具图标与代号 |
| 席面摆位页 | 器具侧栏 | 纵向排列四种器具按钮，选中态高亮边框，已放置显示✓ |
| 席面摆位页 | 合法性校验 | 三条规则逐项显示，合规绿色✓，违规红色✗+说明 |
| 席面摆位页 | 动线步数 | 大号数字展示步数，辅以路径描述文字 |
| 席面摆位页 | 导出摆位表 | 底部按钮，不可用时灰色，可用时茶色，点击弹出 JSON 结果 |

### 4.3 响应式

- 桌面优先（≥1024px），网格区域居中展示
- 平板（768-1023px）侧栏移至顶部水平排列
- 手机（<768px）全宽堆叠布局，侧栏与面板折叠
